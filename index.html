  • Dro-Mal Detector

    A Novel Method of Android Malware Detection

    The increasing popularity of Android applications accompanies with it an ever-growing field of malware. Hence, the need of the hour is to design efficient malware detection techniques to ensure security. Way to find the malicious APKs detection was done in the space of features and permissions. But a part which will cover both the aspects as features (functions) and permissions, i.e. call graphs. Behaviour of an application using caller-callee relationships. Study aimed to use the functional call graphs of an APK generated as a model to track malware behaviour statically and dynamically. The call graph embeddings and permissions when transformed to reduced dimensional feature-vectors, are then trained with the help of SVC and ANN.
